# 2739.구구단

#### N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.

첫째 줄에 N이 주어진다. N은 1보다 크거나 같고, 9보다 작거나 같다.

출력형식과 같게 N*1부터 N*9까지 출력한다.

``` python
2

#2 * 1 = 2
#2 * 2 = 4
#2 * 3 = 6
#2 * 4 = 8
#2 * 5 = 10
#2 * 6 = 12
#2 * 7 = 14
#2 * 8 = 16
#2 * 9 = 18
```



``` python
num = int(input())
for a in range(1, 10):
    print(num, "*", a,"=",num * a)
```

두가지 실수를 저질렀다. for문을 오랜만에 써봤다고는 하지만.. 어떻게 숫자가 순회할 수 있다고 생각했는지.. range를 이용해 9까지의 순회가능한 숫자열을 만들어줬다. 그리고 숫자와 문자열을 이을 때는 +가 작동하지 않는 다는 것을 배웠다. 숫자와 문자열을 이어줄 때는 , 를 활용해야한다. 또한, ,로 이어진 값들 사이에는 기본적으로 공백이 한칸 들어가기 때문에 직접 " * " 형태로 작성하지 않아도 문제에서 원하는 형식이 된다는 것을 알았다.

각 테스트케이스마다 A+B를 한 줄에 하나씩 순서대로 출력한다.

``` python
5
1 1

40 60
1000 1000
```

``` python

46
505
100
2000
```

sys.stdin.readline이 뭘까..? 

``` python

Hello World!

Hello World!
```

인풋 대신에 인풋처럼 사용하는듯 하다. 맨 오른쪽에 `\n`이  디폴트값으로 붙어있기 때문에 제거하고싶다면 .rstrip()을 써야한다고 한다. 뭐가 참 길고 많다..

# 11022. A+B - 8

#### 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

첫째 줄에 테스트 케이스의 개수 T가 주어진다.

각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

각 테스트 케이스마다 "Case #x: A + B = C" 형식으로 출력한다. x는 테스트 케이스 번호이고 1부터 시작하며, C는 A+B이다.

``` python
import sys
N = int(sys.stdin.readline().rstrip())

for i in range(1, N+1):
    print(f'Case #{i}: ', end="")
    A , B = map(int, sys.stdin.readline().rstrip().split())
    print(A, "+", B, "=", A+B)
```

이번에는 한번에 맞았다!! readline()의 괄호를 빼먹지 않고, 문자열과 숫자열을 더하려면 print에서 ,를 사용해야한다는 점, 그리고 ,를 사용해서 묶이게 되면 스페이스 한칸을 기준으로 묶인다는 것을 기억해야겠다. 참고로 문자열은 +로 묶이는데 이때는 띄어쓰기가 적용되지않는다. 

아 그리고, import sys 까먹지말자

# X보다 작은 수

둘째 줄에 수열 A를 이루는 정수 N개가 주어진다. 주어지는 정수는 모두 1보다 크거나 같고, 10,000보다 작거나 같은 정수이다.

``` python
10 5
1 10 4 9 2 3 8 5 7 6
```

``` python
1 4 2 3
```
